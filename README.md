# Math_Typical

「高校数学の問題集 〜最短で得点力を上げる典型250問〜」（マスオ著）の学習補助ウェブアプリ。
スマホ／PC両対応、Googleログインで進捗・メモをクラウド同期する自分専用ツール。

## できること

- 250問を **単元別／難易度別／文理別** にフィルタして閲覧
- 各問題に **ステータス**（未着手 / 解いた / 復習中 / マスター）を付ける
- 各問題に **履歴付きメモ**（解いている間は過去メモを隠せる）
- **解答ボタン**で解答画像をその場で開く
- 進捗・メモは Firestore に同期、画像は Firebase Storage の本人領域に保存

## 設計

学習アルゴリズム・データモデル・UI設計の詳細は `.company/secretary/notes/2026-05-19-decisions.md` に集約。

## ディレクトリ構成

```
Math_Typical/
├── data/
│   └── problems.json     ← 250問のメタデータ（番号・単元・難易度・科目・ページ）
├── scripts/
│   ├── extract_metadata.py  ← PDF からメタデータ抽出
│   └── build_images.py       ← PDF から問題/解答画像を生成
└── images/               ← 画像は .gitignore 対象（Firebase Storage 経由で管理）
    ├── questions/         ← q-001.jpg 〜 q-250.jpg（1問1ページ）
    └── answers/            ← page-037.jpg 〜 page-491.jpg（解答ページ）
```

## ソース PDF（著作物、リポには含めない）

- 印刷用版（1問1ページ）→ 問題画像のソース
- PC版（全492ページ）→ 解答画像のソース
- 配布元: 高校数学の美しい物語

> 本ファイル（ソースPDF）の複製・改変・転載・再配布及び転送は禁止されています。
> 画像も同様に、本人のFirebase Storage領域に保存し、公開リポには含めません。
